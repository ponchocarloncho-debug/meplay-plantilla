ğŸ”„ How It Works

1. Configuration (template.js)

Edit once, applies to all pages:

  • Site name and logo
  • Ad script (injected in <head>)
  • Navigation buttons
  • Search bar
  • Footer

2. Video Database (videodata.json)

Single source of truth for all videos:

  • Powers the home page grid
  • Powers the search functionality
  • Powers the widget recommendations
  • Used by page generators

3. Each Page Loads

Every HTML page includes these scripts:

<script src="template.js"></script>  <!-- Header/footer -->
<script src="search.js"></script>    <!-- Search -->
<script src="widget.js"></script>     <!-- Widgets -->
